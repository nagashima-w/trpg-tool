# CoC 6版→7版 シナリオコンバーター

クトゥルフの呼び声（CoC）第6版用シナリオを、第7版でプレイできるよう自動変換するデスクトップアプリ。

---

## 概要

- **変換方向**: 6版 → 7版のみ（入力は常に6版として扱う）
- **対象**: シナリオ内のNPC・モンスターのステータス
- **形式**: Electronデスクトップアプリ

---

## 入力・出力

| 項目 | 内容 |
|------|------|
| 対応入力形式 | `.txt`、`.pdf` |
| 文字コード | UTF-8、Shift-JIS（自動判別） |
| 出力形式 | `.txt` |
| 変換根拠メモ | UIで確認可能、出力ファイルには含めない |

---

## statブロック検出

以下の能力値キーワードが **3つ以上** 近接して出現する行・段落を検出する。

```
STR / CON / DEX / APP / POW / SIZ / INT / EDU / MOV
HP / MP / SAN
```

**対応フォーマット:**

```
STR 14  CON 12  SIZ 15  INT 7  POW 13  DEX 11     ← スペース区切り
STR:14  CON:12  SIZ:15                             ← コロン区切り
STR：14　CON：12　SIZ：15                           ← 全角コロン・全角スペース
```

**対象外（今バージョン）:** Markdown表形式

---

## 変換ルール

### 能力値

| 対象 | 変換 |
|------|------|
| STR / CON / DEX / APP / POW / SIZ / INT / EDU | × 5 |
| MOV | DEX・STR と SIZ の比較で再計算（下記） |
| HP | (CON×5 + SIZ×5) ÷ 10 切り捨て ※6版と同値 |
| MP | POW×5 ÷ 5 ※6版と同値 |
| SAN | POW×5 ※6版と同値 |

**MOV 再計算（×5後の値で判定）:**

| 条件 | MOV |
|------|-----|
| DEX も STR も SIZ 未満 | 7 |
| DEX も STR も SIZ を超える | 9 |
| それ以外 | 8 |

### ダメージボーナス / ビルド

×5後の STR＋SIZ をもとに7版テーブルで再計算する。

| STR＋SIZ | DB | ビルド |
|----------|----|--------|
| 〜64 | −2 | −2 |
| 65〜84 | −1 | −1 |
| 85〜124 | 0 | 0 |
| 125〜164 | +1D4 | 1 |
| 165〜204 | +1D6 | 2 |
| 205〜284 | +2D6 | 3 |
| 285〜364 | +3D6 | 4 |
| 365〜444 | +4D6 | 5 |
| 445〜 | +5D6 | 6 |

## UI（差分ビュー）

```
┌────────────────────────────────────────────────────────────┐
│  CoC 6→7版 シナリオコンバーター  [ファイルを開く]  [設定]    │
├─────────────────────────┬──────────────────────────────────┤
│  元の文書（6版）          │  変換後（7版）                    │
│                          │                                  │
│  STR 14  CON 12  ...     │  STR 70  CON 60  ...             │
│  ドッジ 22%              │  回避 22%                        │
│  組み付き 55%            │  近接戦闘（格闘）55%              │
│                          │    ↑ クリックで手動編集           │
├─────────────────────────┴──────────────────────────────────┤
│  変換: N件  リネーム: N件  [AIでバランス確認]  [txt出力]     │
└────────────────────────────────────────────────────────────┘
```

- 左右並列表示
- 変更された値・技能名はハイライト表示
- 変換後パネルの値・技能名はクリックで手動編集可能
- statブロック以外のテキストは両パネルで同じ

## AI連携（将来対応）

戦闘バランス・呪文/POW消費の7版的妥当性チェックをAIで支援する機能。

- 対応サービス: Claude API、Gemini API（ユーザーが選択）
- APIキーは設定画面でユーザーが入力（ローカル保存）
- ユーザーが明示的に「AIでバランス確認」ボタンを押したときのみ呼び出し
- AI提案は変換後パネルに追記。採否はユーザーが選択

**今バージョンでは未実装。拡張ポイントのみ残す。**
